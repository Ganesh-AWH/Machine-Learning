🔍 Analyzing Missing Values¶

One of the first steps in data preprocessing is to identify and handle missing values. Let's check which features have missing values and visualize them.

Key Concepts:

  • Missing Completely at Random (MCAR): Missing values don't depend on any values in the dataset
  • Missing at Random (MAR): Missing values may depend on observed values but not on unobserved values
  • Missing Not at Random (MNAR): Missing values depend on unobserved values

🔢 Encode Categorical Variables¶

We need to convert categorical columns—like Sex and Embarked—into numbers that machine learning algorithms can understand. one-hot encoding to:

  • Create new columns representing each category (except one to avoid redundancy)
  • Drop the original Sex and Embarked columns
  • Concatenate the new columns back to the DataFrame
In [17]:
from sklearn.preprocessing import OneHotEncoder

# Encode 'Sex' and 'Embarked' using one-hot encoding
encoder = OneHotEncoder(sparse_output=False, drop='first')
encoded_sex_embarked = encoder.fit_transform(df[['Sex', 'Embarked']])
encoded_df = pd.DataFrame(
    encoded_sex_embarked,
    columns=encoder.get_feature_names_out(['Sex', 'Embarked'])
)

# Remove the original columns and add the encoded columns
df.drop(columns=['Sex', 'Embarked'], inplace=True)
df = pd.concat([df, encoded_df], axis=1)
